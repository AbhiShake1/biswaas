# Biswaas (विश्वास)

Focused Nepal business discovery and review experience.

## Tech Stack

- **SvelteKit 2** + Svelte 5 (runes)
- **Convex** — backend queries and schema
- **WorkOS AuthKit** — authentication
- **Tailwind CSS v4** — styling

## Current Scope

- Business listing by category
- Search by area
- Business details
- Ratings summary
- Customer comments and review submission

## Development

```sh
npm install
npm run dev
```

## Convex

```sh
npx convex dev
```
