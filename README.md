# Biswaas (विश्वास)

Nepal's Trust & Review Platform — like Trustpilot, built for Nepal.

## Tech Stack

- **SvelteKit 2** + Svelte 5 (runes)
- **Convex** — real-time database
- **WorkOS AuthKit** — authentication
- **Tailwind CSS v4** + shadcn-svelte
- **Tauri v2** — desktop app

## Categories

1. Education Consultancies
2. E-Commerce & Online Shopping
3. Trekking & Tourism
4. ISPs & Telecom
5. Hospitals & Healthcare
